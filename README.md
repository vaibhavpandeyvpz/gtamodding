# GTA Modding Tools

A collection of professional tools for modding Grand Theft Auto games.

## Tools

- **Gexter** - GXT file viewer and editor
- **GTA Handler** - Visual editor for handling.cfg files
- **GTA IMG Tool** - IMG archive viewer and editor
- **TXD Edit** - Cross-platform texture dictionary editor
- **Unpaker** - Pak archive tool for Definitive Edition games

## GitHub Pages Setup

This site is designed to be hosted on GitHub Pages. To enable it:

1. Go to your repository settings
2. Navigate to "Pages" in the left sidebar
3. Under "Source", select the branch you want to use (typically `main` or `master`)
4. Select the `/ (root)` folder
5. Click "Save"

The site will be available at `https://vaibhavpandeyvpz.github.io/gtamods/`

## Local Development

To view the site locally, you can:

1. Use a simple HTTP server:
   ```bash
   python3 -m http.server 8000
   ```
   Then open `http://localhost:8000` in your browser

2. Or use any other static file server of your choice
